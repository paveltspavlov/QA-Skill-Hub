How to Use These Skills

Each card below is a complete, ready-to-use skill. To install one in your project, copy the entire card content and save it as a Markdown file inside your repo's .copilot/skills/ folder.

1

Pick a Skill

Browse the QA and Dev sections below. Click any card to expand it and reveal the full skill body — Role, prompt, deliverable shape, and quality checklist.

2

Copy the Entire Card

Select all the content shown when the card is expanded — from the title down to the last line. Don't copy just the prompt block; the agent needs the full structure to behave correctly.

3

Save to .copilot/skills/

In your project root, create the folder .copilot/skills/ if it doesn't exist. Paste the copied content into a new file named <skill-id>.md (e.g. test-case-generator.md).

4

Invoke from Your AI Tool

Open Copilot Chat, Claude Code, or any AI assistant that reads your repo. Reference the skill by name or paste the prompt block. Always review the output before committing — AI output is a starting point, not a final answer.
