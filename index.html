Using Agent Skills Claude Code & Copilot

Agent skills work differently from prompt templates. Instead of copy-pasting, they run autonomously inside your IDE — reading your code, executing commands, and writing files directly.

1

Add Config to Your Project

Copy the "Configuration hints" from any agent skill into your project's CLAUDE.md (Claude Code) or .github/copilot-instructions.md (Copilot). This teaches the agent your project's conventions.

2

Trigger Naturally

Just describe what you need in plain English. The agent reads your diffs, runs commands, and applies the skill's logic automatically — no prompt engineering required.

3

Review Changes

The agent writes files and runs commands on your behalf. Always review the diff before committing. Use git diff to verify changes match your expectations.

4

Iterate & Customize

Refine your CLAUDE.md rules over time. The more specific your project conventions, the better the agent performs. Treat it like onboarding a junior developer.
